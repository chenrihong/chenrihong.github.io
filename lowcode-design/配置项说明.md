# 配置项说明

## hideFlag

 `string`

是否隐藏（表单区+条件区）

| ID  | 名称     |
| --- | ------ |
| 0   | 显示     |
| 1   | 隐藏     |
| 2   | 始终隐藏   |
| 3   | 新增时隐藏  |
| 4   | 非新增时隐藏 |

## useFlag 

`nubmer`

显示位置（条件区）

| ID  | 名称   |
| --- | ---- |
| 0   | 更多   |
| 1   | 常用   |
| 2   | 表格顶部 |
| 3   | 表格左侧 |
| 4   | 表格右侧 |
