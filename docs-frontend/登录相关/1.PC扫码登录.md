# 1.PC扫码登录

:::tip[企微、钉钉扫码登录]
1. 先配置好消息应用
2. 开启参数
:::

## 接口定义

### 1. PC端登录界面获取二维码接口

入参：
- appId：应用ID
- hospId: 医院ID
- tenantId：租户ID

出参：
- status: 状态码
    - 1: 表示成功，返回 uuid
    - 2: 表示未找到应用信息
    - 3：表示未找到租户医院信息
- uuid
- appType: 1-钉钉 2-企微

### 2. PC端轮询获取登录结果接口

`轮询接口，每 5 秒调用一次`

入参：
- uuid

出参：
- status: 状态码
    - 1: 表示登录成功，返回正常登录时的登录结果
    - 2: 表示登录失败，返回失败信息
- message: 登录失败信息

### 3. 手机端扫码登录接口

手机自动登录，调用原有登录接口。登录成功后，会调接口，后端判断uuid是否正常

入参：
- uuid

出参：
- status: 状态码
    - 1: 表示登录uuid正常，允许登录
    - 2: 表示登录失败，可能是uuid已过期
- message: 提示信息
