# 02.下载ZIP按钮

:::danger[不支持流程按钮哦]
移动端目前不支持，配置时请选择仅PC
:::


## 函数名

| 函数名 | 功能 |
| ---- | ---- |
| this.downloadZip | 按钮上使用 |

## 参数

```js
export interface DownloadZipArgs {
    url: string,
    data: Record<string, string>,
    fileName?: string
}
```

### 示例

```js
const userInfo = this.getUserInfo();
const hospId = userInfo.hospId;
const tenantId = userInfo.tenantId;

const rows = this.getGridSelectedRows();
if (rows.length === 0) {
    window.message.pcbox.alert('请选择要下载的图片');
} else {
    const ids = rows.map(row => row.file_id);

    this.downloadZip({
        url: '/a0/files/filePic/public/exportPicturesZip',
        data: { hospId, tenantId, fileIds: ids },
        fileName: '测试.zip'
    })
}
```
