## 函数名

| 函数名 | 功能 |
| ---- | ---- |
| validateForm | 按钮上使用 |


## 示例

```js
this.validateForm({ formid: 'xxxxxxxxxxxxxxx'}).then(res=>{
  if(!res.flag){
      // 表单校验未通过
  }
})
```
