# 01.保存按钮函数

:::tip[保存按钮]
:::


## 函数名

| 函数名 | 功能 |
| ---- | ---- |
| saveFormRule | 按钮上使用 |


## 参数

```ts
interface SaveFormRuleOptions {
    formid?: string;
    andAdd?: boolean;
    gid?: string;
    callback?: any;
    editAction?: boolean;
    notCloseForm?: boolean; // 保存成功后不关闭表单
    updateLinkParam?: boolean;
    notValidChange?: boolean;
    isCloseTabs?: boolean;
}
```

```js
this.saveFormRule(args: SaveFormRuleOptions)
``` 


## 示例

1. 保存时不关闭表单界面

```js
this.saveFormRule({
    notCloseForm: true
})
```
