# 10.删除按钮脚本区别

:::danger[删除按钮（关闭按钮）脚本区别]
this.cancel() vs this.deleteCancel()
:::


## 函数名

| 函数名 | 功能 |
| ---- | ---- |
| this.cancel | 按钮上使用 |
| this.deleteCancel | 按钮上使用 |
 
可关闭当前弹窗/页签，支持绑规则

## 区别

当表单以页签方式打开时，**关闭页签时**，会去找关闭按钮并执行相关代码：

- this.cancel() 会触关闭按钮上的规则
- this.deleteCancel() **不会**触发按钮上的规则
