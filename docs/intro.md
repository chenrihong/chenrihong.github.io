# 参与文档建设

:::tip[为何要进行文档建设]

软件文档不仅是开发过程中必要的工具，更是确保软件质量、可维护性和可持续性的关键因素。它从需求、设计、开发、测试、部署和维护等多个环节为团队和用户提供支持，是软件生命周期中不可或缺的一部分。一个好的软件文档不仅能够提高开发效率，还能为软件的长远发展奠定坚实基础。

:::


参与文档建设，你可以通过以下方式：

1. 点击文档下方的“ Edit This Page ” 链接
2. Fork 文档仓库
3. 修改文档
4. 提交 PR
5. 等待审核

等文档审核通过后，文档会自动更新。
