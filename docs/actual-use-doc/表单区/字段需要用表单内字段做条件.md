# 字段搜索时用参数过滤结果


:::tip[表单输入项是科室，当搜索科室关键字时，我需要库位字段过滤科室？]
比如，出库业务中，选择出库科室时，要用当前库位过滤科室条件。
:::

## 配置示例

1. 科室字段编辑方式修改为**数据集选择器**
    
    用下拉组件做搜索，只能在结果中搜索匹配，不能用别的字段过滤数据集。

2. 在科室字段的脚本中添加以下代码

```javascript
this.bringValueToCondition({
  formid:"000120221111065205PWBGHYKT",
  config:[{
    colid:'POSITION_ID',
    tocolid:'POSITION_ID'
  }]
})
```

3. 配置参数说明

- formid
  
  当前表单ID

- config

  - colid: 表单字段
  - tocolid: 条件字段
